メール通知の設定方法

  1. Resend または SendGrid のアカウントを作成
  2. API キーを取得
  3. Cloudflare Workers にシークレットを設定:
    wrangler secret put RESEND_API_KEY
  4. キーワード追加時に「メール通知を有効にする」にチェック

推奨サービス:
• Resend - 月3,000通まで無料
• SendGrid - 1日100通まで無料
