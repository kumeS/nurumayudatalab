# EC販売向け生成AIプロンプト作成ツール

EC販売に特化した生成AI用の定型プロンプトを作成するためのWebツールです。Amazon商品ページ作成、1688店舗評価、レビュー分析、アパレル画像生成など、テンプレート化されたプロンプトに変数を挿入して、最適化されたプロンプトを生成できます。

## 🎯 プロジェクト概要

このツールは、中国の1688商品情報を元に、**Amazon.co.jp向けの最適化された商品ページコンテンツ**や**1688サプライヤーの評価分析**、**購入者レビューを活用した商品ページブラッシュアップ**、**アパレル商品の画像生成プロンプト**のためのプロンプトを作成します。

### 主な特徴

- ✅ **4つの専用プロンプトテンプレート**: Amazon商品ページ作成、1688店舗評価、レビュー分析ブラッシュアップ、アパレル画像生成
- ✅ **変数入力**: 商品名、バリエーション、仕様、レビューなどを入力フィールドに記入
- ✅ **中国語対応**: 1688からのコピペで中国語入力も可能
- ✅ **レビュー分析**: 購入者レビューから顕在・潜在ニーズを抽出
- ✅ **画像生成プロンプト**: ターゲット属性を選択してアパレル商品の画像生成プロンプトを作成
- ✅ **ブランク対応**: 変数が空欄でもプロンプト生成可能
- ✅ **タブ管理**: 複数のプロンプトテンプレートを管理（拡張可能）
- ✅ **ポップアップ出力**: 生成されたプロンプトをモーダルで表示
- ✅ **クリップボードコピー**: ワンクリックでプロンプトをコピー
- ✅ **自動保存機能**: 入力内容を5秒ごとに自動保存（ローカルストレージ）
- ✅ **大きな文字サイズ**: ゴシック体で読みやすい18pxのベースフォント
- ✅ **レスポンシブデザイン**: PC・タブレット・スマホに対応
- ✨ **v1.5.0 リアルタイム文字数カウンター**: 各入力欄の文字数を即座に表示（3000文字以上でオレンジ、5000文字以上で赤表示）
- ✨ **v1.5.0 入力プレビュー機能**: 生成前に入力内容を確認できるプレビューボタン
- ✨ **v1.5.0 入力進捗インジケーター**: フォームの入力完了度を%で表示する進捗バー
- ✨ **v1.5.0 ヘルプツールチップ**: 入力ラベルの?アイコン、ヘッダーバッジにホバーで詳細説明を表示
- ✨ **v1.5.0 キーボードショートカット**: Ctrl+Enterでプロンプト生成、Escでモーダル/プレビューを閉じる
- ✨ **v1.5.1 AIサービス直接リンク**: 生成したプロンプトをChatGPT、Gemini、Claudeで即座に使用可能
- 🆕 **v1.6.0 複数店舗比較機能**: 1688店舗評価で最大3店舗を同時比較評価（店舗数を選択、動的フォーム表示）
- 🆕 **v1.6.0 タブ記憶機能**: ページリロード後も最後に選択していたタブが表示される
- 🆕 **v1.6.0 ウィンドウサイズ記憶機能**: ブラウザウィンドウのサイズを記憶して復元

## 🚀 現在実装済みの機能

### 1. Amazon商品ページ作成プロンプト

最適化されたプロンプトテンプレートを実装：

#### 入力変数

1. **商品名**
   - **コピー元**: 1688商品ページの「商品タイトル」部分
   - **入力内容**: 1688の商品名をそのまま、または日本語に訳して入力
   - **言語**: 日本語・中国語どちらでもOK
   - **例**: レディース ワイドパンツ ハイウエスト 春夏 カジュアル

2. **商品バリエーション**
   - **コピー元**: 1688商品ページの「规格选择」（バリエーション選択）部分
   - **入力内容**: **Amazonで販売予定のバリエーションのみ**を入力（販売しないバリエーションは除外）
   - **含まれる情報**: カラー、サイズ、材質、パターンなど
   - **例**: カラー: ブラック、ベージュ、グレー / サイズ: S、M、L、XL

3. **仕様・説明文など**
   - **コピー元**: 1688商品ページの「商品详情」（商品詳細）部分全体
   - **取得方法**: 折りたたまれている部分も展開してからコピー
   - **含まれる情報**: 素材、サイズ詳細、機能、特徴、使用方法、注意事項など
   - **例**: 素材: ポリエステル85%、スパンデックス15% / 機能: ストレッチ性、速乾性、通気性 / 特徴: ハイウエストデザイン、ゆったりシルエット、オールシーズン対応

#### 出力内容
プロンプトには以下の項目が含まれます：

1. **商品タイトル**（80〜110文字）
   - ビッグキーワードを前方配置
   - 需給比の高いキーワード優先
   - 単語の重複を回避

2. **検索キーワード**（250バイト以内 = 全角83文字以内）
   - タイトルで使用した単語は除外
   - ビッグキーワードとニッチキーワードのバランス

3. **Bullet Points（箇条書き）**
   - 5つの特徴（各250〜500文字）
   - `<br>`タグで改行

4. **商品説明文**
   - 5つの箇条書き（各250〜500文字）
   - 通常の改行を使用
   - 【】や「」で強調

5. **ビッグキーワード10個**
   - 半角スペース区切り
   - 単語のみ（複合語なし）

6. **ニッチキーワード10個**
   - 半角スペース区切り
   - 単語のみ（複合語なし）

### 2. 1688店舗評価プロンプト

日本で販売する商品の仕入れ先候補の選定を目的とした、1688サプライヤーの総合評価プロンプトを実装：

#### 🆕 v1.6.0 複数店舗比較機能

**店舗数選択**: 1店舗（単独評価）、2店舗（比較評価）、3店舗（比較評価）から選択可能

- **1店舗モード**: 単独サプライヤーの詳細評価
- **2〜3店舗モード**: 複数サプライヤーを横並びで比較評価

#### 入力変数

##### 単独評価（1店舗）の場合

1. **商品ページ情報**
   - **取得方法**: 1688商品ページ全体（Ctrl+A → Ctrl+C → Ctrl+V）
   - **含まれる情報**: 価格、MOQ、仕様、説明文、配送オプション、カスタマイズ対応など

2. **会社概要ページ（公司档案/工厂档案）**
   - **取得方法**: 店舗ページ上部の会社名をクリック → 表示される会社情報ページで Ctrl+A → Ctrl+C → Ctrl+V
   - **含まれる情報**: 
     - 成立年数（何年営業しているか）
     - 従業員数
     - 登録資本金
     - 工場面積
     - 年間生産能力
     - 主要取引先
     - 輸出実績
     - 認証・資格（ISO、CE、FDAなど）

3. **サプライヤー情報（ホバー表示）**
   - **取得方法**: 商品ページ内の会社名にマウスカーソルを乗せる（ホバー） → ポップアップで表示される情報をコピー
   - **含まれる情報**: 
     - 店舗評価スコア（★評価）
     - 取引成功件数
     - リピート率（回頭率）
     - 返金・紛争率
     - 平均納期
     - レスポンス時間（問い合わせ返信速度）
     - 実力商家バッジ（認証マーク）

##### 比較評価（2〜3店舗）の場合
各店舗（店舗A、店舗B、店舗C）について上記3つの情報を入力：
- **商品ページ情報**: 各店舗の1688商品ページ全体
- **会社概要ページ**: 各店舗の公司档案/工厂档案
- **サプライヤー情報**: 各店舗の会社名ホバー表示情報

#### 出力内容

##### 単独評価プロンプト（1店舗）

1. **商品情報の分析**
   - 数量ごとの価格
   - MOQ（最小発注量）
   - 素材・仕様
   - 商品評価と販売実績
   - 仕入れに際しての懸念点

2. **会社情報と信頼度指標（表形式）**
   - 営業年数（業界平均との比較）
   - 事業形態（工場 or 商社）
   - 従業員数
   - 総合評価
   - 返品率
   - リピート率
   - 取引実績
   - 各指標の平均値とのギャップ分析

3. **総合評価と推奨度**
   - 仕入れ先としての総合評価（S/A/B/C/D）
   - 推奨度とその理由
   - リスク要因
   - 注目すべきポイント

##### 比較評価プロンプト（2〜3店舗）

1. **商品条件の比較（表形式）**
   - 数量ごとの価格、MOQ、配送オプション、決済方法
   - カスタマイズ対応、サンプル提供、OEM/ODM対応

2. **会社信頼度指標の比較（表形式）**
   - 成立年数、従業員数、登録資本金、工場面積
   - 年間生産能力、主要取引先、輸出実績、認証・資格

3. **1688プラットフォーム評価の比較（表形式）**
   - 店舗評価、取引実績、リピート率、返金・紛争率
   - 平均納期、レスポンス時間、実力商家認証

4. **品質管理体制の比較（表形式）**
   - 品質検査体制、不良品対応、アフターサービス、品質保証期間

5. **コスト競争力の比較（表形式）**
   - 単価競争力、大量発注時の価格、送料コスト、総合コストパフォーマンス

6. **各店舗の総合評価**
   - 総合評価（S/A/B/C/D）
   - 推奨度（5段階）
   - 強み（3点）
   - 懸念点（3点）
   - 適した取引形態

7. **最終推奨**
   - 最も推奨する仕入れ先と理由
   - 次点候補と選択条件
   - リスク管理の提案
   - 初回取引の具体的提案

※ 各指標の平均値は1688データ分析ツール（1688数据分析、阿里指数、生意参谋）を参照

### 3. レビュー分析ブラッシュアッププロンプト

購入者レビューを分析して商品ページを最適化するプロンプトを実装：

#### 入力変数
- **商品バリエーション**: この商品が持つバリエーション情報
- **購入者レビュー**: 競合商品や類似商品の購入者レビュー
- **現在のAmazon商品ページ**: 現在のBullet Points・商品説明文

#### 出力内容
プロンプトには以下のブラッシュアップ指示が含まれます：

1. **レビュー分析の注意点**
   - 商品の**弱み**を**強み**に変換
   - レビューは他セラーの商品（バリエーション不一致に注意）
   - レビュー内容を商品説明に適切な用語に変換

2. **Bullet Points（箇条書き）**
   - 5つの箇条書き（各250〜500文字）
   - `<br>`タグで改行
   - 現在の分量を**1.5倍**に増量
   - レビューから得られた強みを反映

3. **商品説明文**
   - 5つの箇条書き（各250〜500文字）
   - 通常の改行使用（`<br>`タグ不要）
   - 現在の分量を**1.5倍**に増量
   - 【】や「」で見出しを強調

4. **購入者ニーズ分析**
   - 顕在ニーズ：購入者が明確に求めているもの
   - 潜在ニーズ：購入者が無意識に求めているもの
   - ニーズ充足要因：何によって要求が満たされているか

5. **レビュー由来の検索キーワード100個**
   - ビッグキーワード
   - 中ボリュームキーワード
   - ニッチキーワード
   - 半角スペース区切り、単語のみ（複合語なし）

### 4. アパレル画像生成プロンプト

アパレル商品の画像生成プロンプトを自動作成：

#### 出力内容

1. **商品の様々な角度（3〜4個）**

   - 動きのあるポーズ

4. **使用シーン・コーディネート（3〜4個）**
   - スタイルに合わせた使用シーン
   - ターゲット属性に適した背景・雰囲気

#### 特徴
- 各プロンプトは**英語**で出力（画像生成AIに最適）

- 性別・年齢層・スタイルに合わせたモデル設定

## 📋 使い方

### Amazon商品ページ作成の場合

1. **タブ選択**: 「Amazon商品ページ作成」タブを選択

2. **商品名を入力**:
   - 1688商品ページの**商品タイトル**をコピー
   - 「商品名」欄に貼り付け
   - 中国語のままでもOK、日本語に訳してもOK

3. **商品バリエーションを入力**:
   - 1688商品ページの**「规格选择」（バリエーション選択）部分**を確認
   - **Amazonで販売予定のバリエーションのみ**を入力
   - 販売しないカラーやサイズは除外
   - 例: カラー: ブラック、ベージュ、グレー / サイズ: S、M、L、XL

4. **仕様・説明文を入力**:
   - 1688商品ページの**「商品详情」（商品詳細）部分**をすべて展開
   - 折りたたまれている部分も開く
   - 全体をコピー（Ctrl+A → Ctrl+C）
   - 「仕様・説明文など」欄に貼り付け
   - **含まれる情報**: 素材、サイズ詳細、機能、特徴、使用方法、注意事項など

5. **プロンプト生成**: 「プロンプトを生成」ボタンをクリック

6. **確認とコピー**: ポップアップで内容を確認 → 「クリップボードにコピー」ボタンをクリック

7. **AI使用**: ChatGPT、Claude、Geminiなどに貼り付けて、最適化された商品ページを生成

### 1688店舗評価の場合

#### 単独評価（1店舗）

1. **タブ選択**: 「1688店舗評価」タブを選択
2. **店舗数選択**: 「1店舗（単独評価）」を選択（デフォルト）

3. **商品ページ情報を取得**:
   - 1688商品ページを開く
   - 折りたたまれている部分をすべて展開
   - Ctrl+A → Ctrl+C でページ全体をコピー
   - 「商品ページ情報」欄に Ctrl+V で貼り付け

4. **会社概要ページを取得**:
   - 店舗ページ上部の**会社名をクリック**
   - 表示される【公司档案】または【工厂档案】ページで Ctrl+A → Ctrl+C
   - 「会社概要ページ」欄に Ctrl+V で貼り付け
   - **含まれる情報**: 成立年数、従業員数、工場面積、認証資格など

5. **サプライヤー情報（ホバー表示）を取得**:
   - 商品ページに戻る
   - 商品ページ内の**会社名にマウスカーソルを乗せる**（クリックしない）
   - ポップアップで表示される情報をコピー
   - 「サプライヤー情報」欄に貼り付け

function exportToCSV(tabId) {
    const inputs = gatherInputs(tabId);
    const prompt = generatePromptForTab(tabId, inputs);
    
    const csvData = [
        ['項目', '内容'],
        ['生成日時', new Date().toLocaleString('ja-JP')],
        ['タブ', getTabName(tabId)],
        ...Object.entries(inputs).map(([key, value]) => [key, value]),
        ['生成プロンプト', prompt]
    ];
    
    const csvContent = csvData.map(row => 
        row.map(cell => `"${cell.replace(/"/g, '""')}"`).join(',')
    ).join('\n');
    
    const blob = new Blob(['\uFEFF' + csvContent], { type: 'text/csv;charset=utf-8;' });
    const link = document.createElement('a');
    link.href = URL.createObjectURL(blob);
    link.download = `prompt_${tabId}_${Date.now()}.csv`;
    link.click();
}
