# Amazon ASIN Performance Tracker

## プロジェクト概要

Amazon ASIN Performance Trackerは、Amazon商品のパフォーマンスを時系列で追跡・分析するWebアプリケーションです。Amazon Business Reportの週次CSVデータを活用して、商品ごとのセッション数、売上、コンバージョン率の推移を視覚化し、パフォーマンスの変化を検知します。

## 主な機能

### 📁 データ管理
- **CSVファイルのドラッグ&ドロップアップロード**: 複数の週次レポートファイルを一括アップロード
- **ファイル日付解析**: ファイル名から自動的に週番号を推測
- **データ検証**: アップロードされたデータの整合性チェック
- **データエクスポート**: 分析結果をCSV形式でエクスポート

### 📊 分析・可視化
- **ASINモニタリング**: 各商品のパフォーマンス状況をカード形式で表示
- **時系列分析**: 上位商品のセッション数推移をグラフで表示
- **異常検知**: 大幅な変化があった商品を自動検出・警告
- **新商品分析**: 新たに登場した商品の初動パフォーマンスを追跡

### 🎯 アラート機能
- **新商品検出**: 新規商品の登場を自動検知
- **急上昇・急落検知**: セッション数の大幅な変化を警告
- **パフォーマンス分析**: 商品ステータスの自動分類（新商品、成長中、急上昇、急落、安定など）

### 📈 統計ダッシュボード
- **総売上高**: 全商品の累計売上額
- **アクティブASIN数**: 最新週でアクティブな商品数
- **新規ASIN数**: 新たに登場した商品数
- **平均成長率**: 全商品の平均パフォーマンス変化率

## 使用方法

### 1. データの準備
Amazon Business Reportから週次データをCSVファイルとしてダウンロードします。
ファイル名は`BusinessReport-DD-MM-YY.csv`形式にしてください。

### 2. データのアップロード
1. アプリケーションを開く
2. ドラッグ&ドロップエリアにCSVファイルをドロップ
3. または「ファイルを選択」ボタンから手動でアップロード

### 3. 分析結果の確認
- **ASINモニタリング**: 商品パフォーマンスの概要
- **時系列分析**: 上位商品の推移グラフ
- **異常検知**: 大幅な変化があった商品一覧
- **新商品分析**: 新商品の初動パフォーマンス

### 4. 詳細分析
商品カードをクリックすると、詳細な時系列グラフが表示されます。

## データフォーマット

CSVファイルは以下の列を含む必要があります：

| 列名 | 説明 |
|------|------|
| （親）ASIN | 商品のASINコード |
| タイトル | 商品名 |
| セッション数 - 合計 | 週間セッション数 |
| ページビュー - 合計 | 週間ページビュー数 |
| 注文された商品点数 | 注文数 |
| ユニットセッション率 | コンバージョン率 |
| 注文商品の売上額 | 売上金額 |
| 注文品目総数 | 注文アイテム数 |
